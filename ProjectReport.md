Project Report: Capstone Money Manager App.
Implementation

In this project, I created a Money Manager App in the framework of HTML, CSS, and JavaScript. I was able to use HTML to determine the construction of the application and CSS to make it responsive and attractive. I wrote the functionality in JavaScript and implemented the concepts of the Object-Oriented Programming to structure the code appropriately. Income and expenses are some of the transactions that one can add, view, edit and delete in the app. Every transaction is saved as an object and is saved in the local storage of the browser such that the information does not get lost when the page is refreshed. I have also introduced such features as financial summary (total income, total expenses, and balance), filtering, and sorting of transactions. I also resorted to event handling and manipulating the DOM to update the interface dynamically in the process of the user doing any action.

Challenges Faced

I faced number of challenges while working on this project. The challenge of dealing with data between the user interface and localStorage was one of the primary challenges, particularly during the CRUD process. This became difficult in ensuring that all changes were captured without the need of refreshing the page. Form validation was also a bit challenging to me and I had to deal with the various conditions such as not allowing empty input, invalid dates and wrong value. Another problem was to design a layout responding to various screen sizes. In addition to this, I found it difficult to comprehend and use Object Oriented Programming in JavaScript but as I used it more, I became better.

Key Learnings

I learnt a lot of valuable ideas in web development, time management, as I was managing other assignments too, through this project. I learned more about JavaScript and particularly the use of classes and objects. I also got to know how to store and retrieve data with the local storage. My level of knowledge in the manipulation of the DOM and event handling greatly increased. Further, I was able to learn how to design responsive and user friendly interfaces using HTML and CSS. It is also through this project that I was able to work on my problem solving skills and have confidence in creating actual world web applications on my own.
